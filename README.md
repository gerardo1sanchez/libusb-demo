# libusb-demo
Demo de integración de la libreria libusb y gtk

El objetivo de este programa demostrativo es simplemente mostrar en un listbox de gtk los
vendor id y product id de los dispositivos usb conectados al equipo.

Para compilar y ejecutarse se requiere la instalación de gtkmm-win32-devel-2.22.0-2.exe en C:\gtkmm

Para ejecutar el programa se debe descargar libusb-demo.exe colocado en la raiz del repositorio en
github, y copiarse en C:\gtkmm\bin
